# CVNova — Resume Optimization Platform

An intelligent, AI-powered resume analysis tool that calculates ATS scores, identifies skill gaps, generates career roadmaps, and creates personalized interview questions.

## 🚀 How to Run

Follow these steps to start the application:

1.  **Navigate to the project directory:**
    ```bash
    cd /Users/pranav/.gemini/antigravity/scratch/ai-resume-analyzer
    ```

2.  **Install dependencies:**
    ```bash
    pip3 install -r requirements.txt
    ```

3.  **Start the Flask server:**
    ```bash
    python3 app.py
    ```

4.  **Open in Browser:**
    Access the application at [**http://localhost:8080**](http://localhost:8080)

## ✨ Features

- **Visual Dashboard**: Real-time progress bars for Resume Score, Job Match, and Skill Gap.
- **Career Roadmap**: Personalized learning timeline based on missing skills.
- **AI Interview Prep**: Tailored questions (Technical, Project-Based, Behavioral).
- **JD Matching**: Detailed analysis of your resume against any job description.
- **30+ Job Roles**: Specialized analysis for various tech and creative roles.
- **Custom Role Support**: Intelligent analysis for any job title you specify.
